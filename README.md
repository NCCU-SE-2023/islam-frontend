# ISLAM-frontend

[Github](https://github.com/NCCU-SE-2023/islam-frontend)

## Requirements
- Nodejs >= 16 (`16.20.0` is recommanded)
- Node Package Manager (`pnpm` is recommanded)

## Installation
```
pnpm install
```

## Development
```
pnpm run dev
```

## Lint
```
pnpm run lint
```

## Test
```
pnpm run test
```
